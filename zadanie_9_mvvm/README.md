# Практическое задание — Мини-фреймворк MVVM

Реализован мини-фреймворк MVVM на Python.

## Структура
- core — базовые механизмы уведомлений
- models — модели данных
- viewmodels — слой логики
- views — представление (tkinter)

## Особенности
- Одна модель может использоваться несколькими компонентами.
- При изменении модели все связанные View автоматически обновляются.
- Реализованы обёртки над визуальными компонентами (BoundEntry, BoundButton).

## Запуск

python app.py
