Задание 7

Реализовано префиксное дерево для хранения слов.
Поддерживаются операции:
	•	insert (word) - вставка слова
	•	search (word) - поиск полного слова
	•	starts_with (pref) - получение всех слов начинающихся с префикса

Файлы
	•	zadanie_7.py -  реализация Trie
	•	test_zadanie_7.py - тесты pytest
	•	README.md - описание и инструкция запуска

Требования
- Python 3.9+

# Установка pytest
pip install pytest

# Запуск тестов
pytest
